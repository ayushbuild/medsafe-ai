# MedSafe AI

MedSafe AI is an AI-powered medicine safety assistant built using Python and Streamlit.

## Features

- Medicine Interaction Checker
- Prescription OCR Reader
- Symptom Solver
- Side Effect Monitor
- Emergency Risk Predictor

## Technologies Used

- Python
- Streamlit
- Fuzzy Matching
- OCR

## How to Run

1. Clone the repository
2. Install requirements

pip install -r requirements.txt

3. Run the app

streamlit run app.py
